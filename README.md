Github setup issues
-------------------
**设置代理服务器goagent绕过GFW**

在Git Shell中设置代理服务器地址：
git config --global http.proxy 127.0.0.1:8087

**解决Certificate error**

在Git Shell 中disable sslverify：
git config --global http.sslverify false
